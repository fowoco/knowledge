# 공식 출처와 갱신 정책

## 우선순위

1. 현행 법령·정부 부처·공공기관의 최신 원문
2. EPS·하이코리아·고용24 등 실제 업무 시스템 안내
3. 공공데이터포털 원본 데이터
4. 공공 연구기관 보고서와 통계
5. 인터뷰·커뮤니티는 문제 가설과 표현 수집에만 사용

블로그와 커뮤니티 글은 법적 절차의 정답으로 사용하지 않습니다.

## 사용 구분

| 목적 | 허용 자료 | 금지 |
| --- | --- | --- |
| Workflow | 공식 절차·서식·필수서류 | 커뮤니티 경험을 공식절차로 일반화 |
| 모델 학습 | 동의·비식별·검수된 문장 | 원본 인사정보·원본 메신저 대화 |
| 모델 보조 | AIHub·KLUE의 일반 Intent/NER | 공개 라벨을 FOWOCO 라벨로 무검수 치환 |
| 평가 | 분리·고정된 Gold 사례 | 학습 또는 Few-shot 예시로 재사용 |

## 갱신

- 공식 출처는 `source_id`를 유지하고 URL·게시일·검증일을 기록
- 법령·서식 변경 시 영향받는 Workflow와 체크리스트를 Issue에 명시
- 두 명 검수 후 Context Pack 버전 증가
- 과거 버전은 삭제하지 않고 Agent 실행 로그에 사용 버전 기록

## 라이선스

- AIHub는 데이터셋별 이용조건과 재배포 제한을 확인
- KLUE NER는 CC BY-SA 4.0 표시와 조건 준수
- 공공데이터는 각 데이터 페이지의 이용조건과 제3자 권리를 확인
- 원본 데이터를 이 저장소에 복제하기보다 출처 레지스트리와 변환 스크립트를 우선 관리
