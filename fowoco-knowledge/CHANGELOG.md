# Changelog

## 0.4.0 - 2026-07-16

- 주체·기한·수량 등 객관적 필수정보를 기준으로 모호성 범주 확장
- 핵심값 누락·변경·근거 없는 추가와 법령 단정·외부 실행을 구분하는 출력 품질 계약 추가
- 의도적 오류 9건을 포함한 안내문 품질 회귀평가 12건과 결정론적 검사기 추가
- Workflow별 허용 Slot만 남기는 LLM 전송 최소화와 직접식별자 마스킹 구현
- 직접 E-9 제조업 근거와 인접 다국어 행정 근거를 분리한 익명 인터뷰 발견 13건 추가
- EPS 용어·외국어 DB 등 외부자료의 적재·보류 기준 문서화

## 0.3.0 - 2026-07-16

- 대표 모델을 Intent·Domain 다중라벨 분류로 고정한 데이터 계약 추가
- Seed·평가 데이터의 역할, 평가 잠금, 증강 금지 정책 명시
- 데이터 분포·중복·분할 누수·개인정보 패턴·준비 상태 리포트 CLI 추가
- 기존 제안 라벨을 숨긴 2인 독립 검수 큐와 최종 합의 양식 추가
- 두 검수 결과의 Exact Match·Kappa와 불일치 큐 생성 CLI 추가
- 팀 분류 라벨과 전문가 Workflow 검수 범위를 분리
- Intent·Domain 경계 사례와 데이터 담당 운영 절차 문서화
